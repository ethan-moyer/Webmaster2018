Costs

The pricing for using an A.I. changes depending on which service and which A.I. is being used. The pricing for the Google Cloud ML Engine changes depending on the type of the machine. The cheapest version is the Basic tier which costs $0.2774 per hour. The most expensive version is the Premium_1 tier which costs $24.1683 per hour. IBM Watson, an A.I. created for businesses has a similar tier system ranging from Lite to Professional. The Lite version is free and gives the user 5,000 A.I. predictions every month, and the Professional version costs $1,000 for each instance and the user gets 2,000 predictions.
